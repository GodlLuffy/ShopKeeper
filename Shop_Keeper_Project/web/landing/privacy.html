Privacy Policy

Effective Date: March 30, 2026

Welcome to ShopKeeper PRO. This Privacy Policy is based on GSD project specifications for the ShopKeeper Retail OS, specifically for Google OAuth compliance.

1. Information We Collect

We collect only necessary information to facilitate retail ERP operations:

  • Identity: Name, email, and shop profile imagery.
  • Metrics: Sales records, inventory stock levels, and expense transactions.
  • Security: 4-digit PIN hashes and biometric authentication metadata.

2. Data Usage & Cloud Sync

We leverage Google Firebase (project: shopkeeper-9f594) for real-time cloud synchronization. Your data is used for:

  • Maintaining account persistence across multiple terminals.
  • Generating AI-driven revenue insights (Profit/Loss).
  • Securing transaction history through Firestore security rules.

3. Transparency

This application follows the Royal Obsidian design system and adheres to the security constraints outlined in the project README. Data is encrypted in transit and at rest using standard Google Cloud protocols.
